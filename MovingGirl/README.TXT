------------------------------------------------------------------------
README for MovingGirl - A demo of a platform jumper game in Greenfoot
------------------------------------------------------------------------

PROJECT TITLE: Moving Girl

PURPOSE OF PROJECT: Demo to show how a platform jumper game can be
created in Greenfoot.

VERSION or DATE: 2014-06-04.

AUTHORS: Thomas Ejnefjäll. Inspired by code from Griff (griff@kvusd.org).

USER INSTRUCTIONS: The girl can move around using left and right arrow 
keys and jump with the up key.
